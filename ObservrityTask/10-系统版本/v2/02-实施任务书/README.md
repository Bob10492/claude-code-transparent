- `00-阶段总路线`
  - 最早的总体实施任务书和执行清单
- `01-V2.1-V2.2`
  - 从手动绑定到自动 runner、V2.2 alpha/beta 相关任务书
- `02-V2.3-V2.5`
  - 当前最值得继续阅读的主线任务书
- `90-历史补丁与过渡稿`
  - 过渡时期的补丁包、临时桥接任务书
